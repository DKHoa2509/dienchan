# Mobile MP3 Player

Static one-screen mobile music player.

## Files

- `index.html` - player markup
- `styles.css` - full-viewport mobile UI
- `script.js` - play/pause, seek, replay, loop state
- `assets/images/anhnen.png` - background image
- `assets/audio/nhacnen.mp3` - looped MP3

No build command is required on Vercel.
